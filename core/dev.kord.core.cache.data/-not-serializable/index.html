NotSerializable

@KordExperimental
object NotSerializable : KSerializer<Any?> (source)

A serializer whose sole purpose is to provide a No-Op serializer for Any. The serializer is used when the generic type is neither known nor relevant to the serialization process

e.g: Choice<@Serializable(NotSerializable::class) Any?> The serialization is handled by Choice serializer instead where we don't care about the generic type.
